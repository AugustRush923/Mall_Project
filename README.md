# MallProject
基于DRF+Vue的前后端分离电商项目

## 电商网站

* 该项目为一款B2C模式的商城项目
* 其页面与功能与京东类似
* 基于DRF+Vue，以**前后端分离**的形式实现具体业务逻辑
